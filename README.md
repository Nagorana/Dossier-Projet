# 1: CresellCard
C'est un site web qui sert au collectionneur et au marchand de cartes de collection.  
On peut y acheter et encherir sur des cartes.  
Il est aussi possible de vendre ses cartes.  
# 2: Installation necessaire
IDE Intellij  
Language Java  
Library Springboot  
Base de donnés My Sql
# Etape 1
Cloner le projet depuis Github ||| git clone https://github.com/Nagorana/Front |||
# Etape 2
Ouvrir L'IDE  
Selectionner le dossier du projet  
# Etape 3
Entrer dans le dossier frontend ||| cd Frontend |||  
Installer les dépendance avec ||| npm install |||  
Demarrer avec ||| npm run |||


# 3: Présentation Générale de la structure
Ce document décrit l'architecture du projet

Il se découpe en 3 parties

Le Frontend:C'est la partie visible de l'utilisateur.

Le Backend et les Api:C'est la logique métier et la gestion des données.

La Base de données:C'est le stockage des informations du projet.
